# Final-Project
putting together my final project

Valid commands:
    go <cardinal direction> - moves to room in that direction
    look - Only refreshes the description of the room
    take <item name> - takes the item and places it in inventory
    talk - Begins a talk loop with the Old Man, but only in his room
    inventory - shows your inventory
    quit - end the game'

Your goal is to get through the cave's twisting corridors.  Explore, find items, and make your way deeper into the cave system.  Will you get out?
